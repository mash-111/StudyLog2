# StudyLog2 - 学習時間記録iOSアプリ

## プロジェクト概要
Swift/SwiftUIで作るiOSネイティブアプリ。
学習時間をタイマーで計測・記録し、項目ごとに分類・統計で振り返れる。

## 技術スタック
- Swift / SwiftUI
- SwiftData（データ永続化）
- UserNotifications（目標達成通知）
- Charts framework（グラフ表示）
- iOS 17以上をターゲット

## 画面構成
1. ホーム画面：タイマー＋今日の学習サマリー
2. 記録一覧：過去の学習ログ
3. 統計画面：グラフ・週/月単位の集計
4. 設定画面：項目管理・目標設定

## コーディング規則
- ViewとロジックはViewModel（MVVM）で分離すること
- SwiftDataのModelは Models/ フォルダに置くこと
- Previewが動くようにサンプルデータを必ず用意すること
- コメントは日本語でOK

## エージェントへの注意
- ビルドエラーは自分で修正してからレポートすること
- 画面遷移は NavigationStack を使うこと
- カラーはアセットカタログで管理すること
